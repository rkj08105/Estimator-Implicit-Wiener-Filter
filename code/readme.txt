
The description of each program file is as follows:

implicit_wiener.m  - Program file for implicit Wiener filter algorithm
spec_sub.m         - Program file for spectral subtraction algorithm
obj_measures.m     - Program file to calculate objective measures
comp_llr.m         - Program file to calculate Log likelihood ratio 
comp_cep.m         - Program file to calculate Cepstrum distance
comp_wss.m         - Program file to calculate Weighted Spectral Slope
SNRseg.m           - Program file to calculate segmental SNR
